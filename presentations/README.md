# Presentations

Summary presentations generated for the Citicorp Center simulation project.

| File | Description | Generator Script |
|------|-------------|-----------------|
| `citicorp_simulation_summary.pptx` | Main simulation overview (all modules) | `scripts/create_slides.py` |
| `citicorp_16scenario_summary.pptx` | 16-scenario Monte Carlo results | `scripts/create_slides.py` |
| `mesh_methods_presentation.pptx` | NYC building data pipeline and mesh methods | `citicorp_cfd/paraview/create_mesh_presentation.py` |
| `cityscape_comparison_v2.pptx` | Old vs new cityscape geometry comparison | `citicorp_cfd/paraview/create_comparison_presentation.py` |
| `un_hq_presentation.pptx` | UN Headquarters CityGML extraction demo | `citicorp_cfd/paraview/create_un_hq_presentation.py` |

## Regenerating

All presentations are generated by Python scripts using the `python-pptx` library:

```bash
pip install python-pptx
python scripts/create_slides.py
```

For CFD-specific presentations, run from the `citicorp_cfd/` directory:

```bash
cd citicorp_cfd
python paraview/create_mesh_presentation.py
python paraview/create_un_hq_presentation.py
```
